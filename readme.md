.admx шаблон групповой политики для управления параметрами безопасности ChmHelp
===============================================================================

Данный шаблон позволяет посредством редактора GPO изменять параметры ChmHelp в соответствии с [kb896054](http://support.microsoft.com/kb/896054).

Развёртывание пакета через GPO
------------------------------

Сохраняем файлы .admx и .adml в %WinDir%\PolicyDefinitions. После этого открываем редактор GPO:

Конфигурация компьютера / Административные шаблоны / Система / Справка .chm
